# Madhumitha M - Portfolio Website

A modern, interactive portfolio website built with React, Vite, and Tailwind CSS, featuring smooth animations and a beautiful pastel green design theme.

## Features

- **Responsive Design**: Fully responsive across all devices
- **Dark/Light Mode**: Theme toggle with smooth transitions
- **Interactive Animations**: Motion-powered animations with proximity-based icon interactions
- **Modern UI**: Glass-morphism effects and gradient backgrounds
- **Sections**:
  - Hero with typing animation and mountain landscape
  - About Me
  - Education with animated sparkles and rotating blobs
  - Skills (Programming, EDA Tools, CAD Tools, Operating Systems, Core Domains)
  - Roles & Activities
  - Projects linking to GitHub repositories
  - Contact information

## Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS v4** - Styling
- **Motion (Framer Motion)** - Animations
- **Lucide React** - Icons
- **Radix UI** - Accessible UI components

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment on Vercel

### Option 1: Deploy via Vercel Dashboard

1. Push your code to a Git repository (GitHub, GitLab, or Bitbucket)
2. Go to [vercel.com](https://vercel.com) and sign up/login
3. Click "Add New Project"
4. Import your repository
5. Vercel will auto-detect the Vite framework
6. Click "Deploy"

### Option 2: Deploy via Vercel CLI

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```
